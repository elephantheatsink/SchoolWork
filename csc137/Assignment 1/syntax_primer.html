Verilog Syntax Primer

When you have multiple source-code files, you can use
include to add them to the current program, e.g.,
   `include "defines.v"
   `include "cpu.v"
   `include "memory.v"
   `include "control_unit.v"

To define constants, e.g.,
   `define WORD_SIZE 32    // "back-tick" symbol, not single quote
   `define LD    5'b00001
   `define ST    5'b00011
   `define ADD   5'b01100

Use parameter to declare constant variables, e.g.,
   parameter word_size = 32;
Multiple definitions:
   parameter
         one = 1,
         two = 2,
         three = 3;

The "define" and "parameter" statements can occur either inside
or outside of "module."
