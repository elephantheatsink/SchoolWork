Background

Terry Fay from Fire Risk Alliance wanted us to build a website that organizes fire
sprinklers and their spray patterns.

Scope

The client needs a standalone webpage where users would log in and have access to a
subset of a database of sprinklers. The customers would be able to view the sprinkler
spray patters that are displayed on the webpage.

Solution

We used the MERN stack (MongoDB, Express, React, and Node.js) for the website.
We used D3.js for the visualization of the sprinkler patterns

Highlights

Our client was interested in doing this project for a long time. He told us
this sort of website did not exist yet, and how his job would be much easier if
it was made.

Implementation

Coding and development for Node.js and react was done mostly in Visual Studio Code.
MongoDB cloud database was managed through MongoDB Compass application.
For version control, we used the GitLab account provided by the school.
To manually test the functionality and view the website, we used various web browsers
such as Chrome, Firefox, and Safari.
Deployment was done with GoDaddy, cPanel, and PuTTY.
The project was built from the ground up, using the React App.
